# Artezio bank app

## Requirements

- Install Node
    - https://nodejs.org/en/



## Installing Bower
`npm install` will install these too, but you can do it manually.
- Open terminal
- Type `sudo npm bower install -g`

## Installing Node.js Packages
- Open terminal in your directory
- Type `npm install`

## Installing Bower Packages
`npm install` will install these too, but you can do it manually.
- Open terminal in your directory
- Type `bower install`

## Running
- Open terminal in your directory
- Type `node bin/http-server`
