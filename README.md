# Java client-server app

## Requirements

- Install Node
    - https://nodejs.org/en/

## Installing Node.js and Bower Packages
- Open terminal
- Type `npm install`

## Installing Bower Packages
`npm install` will install these too, but you can do it manually.
- Open terminal
- Type `bower install`

## Running
- Open terminal
- Type `http-server -p 8000`
